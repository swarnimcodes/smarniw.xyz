# Portfolio Website

This repo contains the code for my personal portfolio website hosted at: [smarniw.xyz](https://smarniw.xyz)

## Technologies Used:
- [Bun](https://bun.sh/) -- package manager
- [AdonisJS](https://adonisjs.com/) -- backend framework to create fullstack apps
- [Inertia](https://inertiajs.com/)
- [React](https://react.dev/)
- [Vite](https://vite.dev/)
- [Tailwind](https://tailwindcss.com/)
